QDomDocument SvgGenerator::mergeSvgDoms ( )
private

Merge all the SVGs in the m_domList list into a single SVG.

Returns
The merged SVG as a DOM document

m_domList contains DOM representations of multiple SVGs Assuming each svg contains a main group containing every graphical item of the svg, this function adds to the first svg of the list all the main groups of the other svgs, thus creating a single svg with each svg of the list in it, every svg being in its own group. This function is used in order to create svgs containing groups
